# AISTATE Web — Informacje

**AISTATE Web** (*Artificial Intelligence Speech‑To‑Analysis‑Translation Engine*) to aplikacja WWW do transkrypcji, diarizacji oraz generowania raportów.

## Najważniejsze możliwości
- Transkrypcja audio (Whisper)
- Diarizacja (podział na mówców)
- Edycja bloków i ręczne poprawki
- Raporty: **TXT / HTML / PDF**
- Ustawienia (np. token HF, modele, język interfejsu)
- Logi działania aplikacji

## Szybki start
1. Uruchom backend i webapp (zgodnie z instrukcją projektu).
2. Otwórz przeglądarkę i wejdź na panel aplikacji.
3. Wybierz plik audio → transkrypcja → diarizacja → eksport raportu.
