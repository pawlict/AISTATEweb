# AISTATE Web — 정보

**AISTATE Web** (*Artificial Intelligence Speech‑To‑Analysis‑Translation‑Engine*)은 **전사**, **화자분리**, **번역**, **GSM/BTS 분석**, **AML 금융 분석**을 위한 웹 애플리케이션입니다.

---

## 🚀 주요 기능

- **전사** — 오디오 → 텍스트 (Whisper, WhisperX, NeMo)
- **화자분리** — "누가 언제 말했는가" + 화자 세그먼트 (pyannote, NeMo)
- **번역** — 텍스트 → 다른 언어 (NLLB‑200, 완전 오프라인)
- **분석 (LLM / Ollama)** — 요약, 인사이트, 보고서
- **GSM / BTS 분석** — 빌링 데이터 가져오기, BTS 지도, 경로, 클러스터, 타임라인
- **금융 분석 (AML)** — 은행 명세서 파싱, 위험 스코어링, 이상 거래 감지
- **로그 및 진행 상황** — 작업 모니터링 + 진단

---

## 🆕 3.6 베타 새로운 기능

### 📱 GSM / BTS 분석
- 빌링 데이터 가져오기 (CSV, XLSX, PDF)
- 다양한 뷰가 포함된 인터랙티브 **BTS 지도**: 포인트, 경로, 클러스터, 이동, BTS 커버리지, 히트맵, 타임라인
- **오프라인 지도** — MBTiles 지원 (래스터 PNG/JPG/WebP + MapLibre GL을 통한 벡터 PBF)
- **오버레이 레이어**: 군사 기지, 민간 공항, 외교 공관 (내장 데이터)
- **KML/KMZ 가져오기** — Google Earth 및 기타 GIS 도구의 사용자 정의 레이어
- 공간 쿼리를 위한 영역 선택 (원형 / 사각형)
- 연락처 그래프, 활동 히트맵, 주요 연락처 분석
- 워터마크가 포함된 지도 스크린샷 (온라인 & 오프라인 지도 + 모든 오버레이 레이어)

### 💰 금융 분석 (AML)
- 은행 명세서를 위한 **자금세탁방지** 파이프라인
- 자동 은행 감지 및 PDF 파싱: PKO BP, ING, mBank, Pekao SA, Santander, Millennium, Revolut (+ 범용 폴백)
- MT940 (SWIFT) 명세서 형식 지원
- 거래 정규화, 규칙 기반 분류 및 위험 스코어링
- **이상 감지**: 통계적 베이스라인 + ML (Isolation Forest)
- **그래프 분석** — 거래 상대방 네트워크 시각화
- 다중 계좌 조사를 위한 교차 계좌 분석
- 지출 분석, 행동 패턴, 가맹점 분류
- LLM 지원 분석 (Ollama 모델용 프롬프트 빌더)
- 차트가 포함된 HTML 보고서
- 안전한 공유를 위한 데이터 익명화 프로필

## 🔐 보안 및 사용자 관리

AISTATE Web은 두 가지 배포 모드를 지원합니다:

- **싱글 유저 모드** — 간소화, 로그인 불필요 (로컬 / 셀프 호스팅).
- **멀티 유저 모드** — 완전한 인증, 권한 부여 및 계정 관리 (50~100명 동시 사용자 대상).

### 👥 역할 및 권한

**사용자 역할** (모듈 접근):
- Transkryptor, Lingwista, Analityk, Dialogista, Strateg, Mistrz Sesji

**관리자 역할:**
- **Architekt Funkcji** — 애플리케이션 설정 관리
- **Strażnik Dostępu** — 사용자 계정 관리 (생성, 승인, 차단, 비밀번호 재설정)
- **Główny Opiekun (슈퍼 관리자)** — 모든 모듈 및 관리 기능에 대한 전체 접근

### 🔑 보안 메커니즘

- **비밀번호 해싱**: PBKDF2-HMAC-SHA256 (260,000 반복)
- **비밀번호 정책**: 설정 가능 (없음 / 기본 / 중간 / 강함); 관리자는 항상 강력한 비밀번호 필요 (12자 이상, 대소문자 혼합, 숫자, 특수 문자)
- **비밀번호 블랙리스트**: 내장 + 관리자 관리 사용자 정의 목록
- **비밀번호 만료**: 설정 가능 (X일 후 강제 변경)
- **계정 잠금**: 설정 가능한 실패 횟수 후 (기본 5회), 15분 후 자동 잠금 해제
- **속도 제한**: 로그인 및 가입 스로틀링 (IP당 분당 5회)
- **세션**: 보안 토큰 (secrets 모듈), HTTPOnly + SameSite=Lax 쿠키, 설정 가능한 타임아웃 (기본 8시간)
- **복구 문구**: 셀프 서비스 비밀번호 복구를 위한 12단어 BIP-39 니모닉 (~132비트 엔트로피)
- **사용자 차단**: 영구 또는 임시, 사유 포함
- **보안 헤더**: X-Content-Type-Options, X-Frame-Options, X-XSS-Protection, Referrer-Policy

### 📝 감사 및 로깅

- 전체 이벤트 로그: 로그인, 실패한 시도, 비밀번호 변경, 계정 생성/삭제, 차단, 잠금 해제
- IP 주소 및 브라우저 핑거프린트 기록

- 관리자를 위한 사용자 로그인 이력 + 전체 감사 추적
