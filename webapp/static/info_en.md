# AISTATE Web — Information

**AISTATE Web** (*Artificial Intelligence Speech‑To‑Analysis‑Translation Engine*) is a web application for transcription, diarization, and report generation.

## Key features
- Audio transcription (Whisper)
- Diarization (speaker segmentation)
- Block editing and manual corrections
- Reports: **TXT / HTML / PDF**
- Settings (e.g., HF token, models, UI language)
- Application logs

## Quick start
1. Start the backend and the web app (see the project instructions).
2. Open the app in your browser.
3. Select an audio file → transcribe → diarize → export a report.
