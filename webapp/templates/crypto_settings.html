Ustawienia Crypto (Blockchain)

Moduł Crypto jest w fazie testów. Import i analiza danych blockchain (BTC/ETH) odbywa się wyłącznie offline — żadne dane nie są wysyłane na zewnętrzne serwery.

OFAC — Sanctioned Addresses
Lista adresów kryptowalutowych objętych sankcjami OFAC (US Treasury). Używana do automatycznego tagowania transakcji z podmiotami na liście SDN.
Znane kontrakty (DeFi / DEX / Bridge)
Baza znanych adresów smart-kontraktów (Uniswap, Aave, Compound, Curve, 1inch, mostki). Używana do automatycznego etykietowania interakcji z protokołami DeFi.
Konfiguracja danych referencyjnych
Adresy OFAC Tornado Cash, Blender.io, inne Kontrakty DeFi Uniswap, Aave, Compound, Curve, 1inch, 0x, mostki Lokalizacja backend/crypto/config/ Format JSON (ręczna edycja lub automatyczna aktualizacja)
Dane referencyjne są przechowywane lokalnie w katalogu backend/crypto/config/. Aby dodać własne adresy — edytuj pliki sanctioned.json lub known_contracts.json.
Obsługiwane formaty importu
Bitcoin
  • WalletExplorer.com — eksport CSV z portfela (automatyczne wykrycie)
  • Blockchair — eksport transakcji CSV
  • Electrum — historia transakcji CSV
  • Ledger / Trezor — eksport CSV z portfela sprzętowego
Ethereum / EVM
  • Etherscan — transakcje normalne, wewnętrzne i tokenowe CSV
  • MetaMask — eksport CSV
Giełdy (CEX)
  • Binance — historia transakcji i handlu (CSV)
  • Coinbase / Coinbase Pro — eksport CSV
  • Kraken — ledger i trade history (CSV)
  • Bybit, KuCoin — eksport CSV
Automatyczne wykrywanie formatu
System automatycznie rozpoznaje format pliku CSV na podstawie nagłówków kolumn. Wystarczy wgrać plik w zakładce Analiza → Crypto — format zostanie wykryty automatycznie.
Obsługiwane formaty: Binance, Coinbase, Coinbase Pro, Kraken, Bybit, KuCoin, Etherscan (normal/token/internal), Blockchair, Electrum, MetaMask, Ledger, Trezor, WalletExplorer, oraz format generyczny (best-effort).
Jeśli plik nie zostanie rozpoznany automatycznie, system użyje parsera generycznego który spróbuje dopasować typowe nazwy kolumn (hash, timestamp, from, to, amount, token).
